TITLE: Keyboard shortcut reference overlay (Ctrl+? or F1)

DESCRIPTION: Global keyboard shortcut help overlay showing all available shortcuts, context-sensitive to current view.

DETAILS:
PROBLEM:
- Users don't know available keyboard shortcuts
- Docs mention Ctrl+K, Space, Escape but no in-app reference
- Power users want to learn shortcuts to speed up workflow
- Medical residents tired post-shift need zero-friction navigation

SOLUTION:
1. Shortcut overlay:
   - Trigger: Ctrl+? or F1 or Shift+?
   - Modal overlay with semi-transparent background
   - Two-column layout: [Shortcut] [Action]
   - Grouped by context: Global, Capture, Review, Navigation

2. Context-aware display:
   - Show all shortcuts, but highlight relevant ones for current view
   - Example: In Review view, emphasize Space, 1-4, Escape
   - In Capture view, emphasize Ctrl+Enter, Tab, Ctrl+S

3. Shortcut categories:
   GLOBAL:
   - Ctrl+K: Command palette
   - Ctrl+F: Search
   - Ctrl+Enter: Floating capture
   - Ctrl+Z: Undo
   - Ctrl+Shift+Z: Redo
   - Escape: Back/Cancel
   - F1 or Ctrl+?: Show shortcuts
   
   CAPTURE:
   - Ctrl+S: Save card
   - Tab: Next field
   - Ctrl+Shift+C: Make cloze
   
   REVIEW:
   - Space: Show answer / Continue
   - 1-4: Manual grade (if visible)
   - Escape: Exit review
   - Left/Right: Navigate cards (if applicable)
   
   NAVIGATION:
   - Alt+1: Capture view
   - Alt+2: Review queue
   - Alt+3: Knowledge bank
   - Alt+4: Settings

4. Interactive features:
   - Search within shortcuts: Filter as user types
   - Click shortcut to dismiss and execute action
   - "Print" button to print/export shortcuts list
   - Settings link: "Customize shortcuts" (future feature)

5. Visual design:
   - Keyboard key styling: <kbd> tags with 3D effect
   - Monospace font for shortcuts
   - Icons next to actions (same as navigation)
   - Footer: "Dismiss: Esc or click outside"

FILES TO CREATE:
- src/components/modals/KeyboardShortcutsModal.tsx
- src/constants/keyboard-shortcuts.ts: Centralized shortcut definitions
- src/hooks/useKeyboardShortcut.ts: Reusable hook for registering shortcuts
- src/components/ui/KeyboardKey.tsx: Styled <kbd> component

INTEGRATION:
- Add "Keyboard Shortcuts" link in Settings
- Add "?" icon in header/footer
- First-time user: Show tooltip "Press F1 for shortcuts"

ACCESSIBILITY:
- Modal should be keyboard-navigable
- Screen reader friendly
- High contrast mode support
